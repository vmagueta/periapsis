# Contributing to periapsis

Thanks for your interest in contributing! Here's how to get started.

## Getting started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/periapsis.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Run the checks before committing:

```bash
cargo fmt       # Format code
cargo clippy    # Lint
cargo test      # Run tests
```

## Pull requests

- Keep PRs focused on a single change
- Follow Rust conventions and idiomatic patterns
- Run `cargo fmt` and `cargo clippy` with no warnings
- Add tests for new functionality
- Update the README and CHANGELOG if needed

## Code of conduct

Be kind, be respectful. We're all here to learn and build.
